# React Track (학습 기록)

이 폴더는 React를 **강의 3개 트랙**으로 나눠서 학습한 내용을 기록합니다.  
목표는 “문법을 아는 것”이 아니라, **원리 이해 + 실습 프로젝트 + 공식 권장 패턴 정착**까지 가는 것입니다.

---

## 📌 트랙 구성 (3개 강의)

### 1) `01-onebite-react/` (한입 크기로 잘라먹는 리액트) ✅ 실습 중심

- 빠르게 만들면서 React 감각을 잡는 트랙
- 목적: **컴포넌트/상태/훅/비동기**를 손에 익히기

추천 방식

- 섹션 단위로 `apps/`를 분리해서 실습
- 미니 프로젝트 → 최종 프로젝트까지 완주

📂 바로가기: [01-onebite-react](./01-onebite-react)

---

### 2) `02-kimjeonghwan-react/` (김정환: 만들고 비교하며 학습하는 React) 🔥 원리/구현 트랙

- React를 “사용”하는 것을 넘어, 내부 동작을 이해하는 트랙
- 목적: 렌더링/상태 업데이트/구조를 **JS 관점에서 해석**하기

추천 방식

- `labs/`에 작은 실험 코드로 쪼개서 기록
- “완벽 구현”보다 **관점 교정**이 목표

📂 바로가기: [02-kimjeonghwan-react](./02-kimjeonghwan-react)

---

### 3) `03-react-docs-study/` (공식문서 스터디) ⭐ 정리/레퍼런스 트랙

- 공식 문서 기반으로 개념을 최종 확정하는 트랙
- 목적: 내가 쓰는 패턴을 **공식 권장 흐름으로 재정렬**하기

추천 방식

- `notes/`에 짧고 명확한 요약 + 내 코드 예시 링크
- “필요할 때 다시 찾아볼 수 있게” 정리

📂 바로가기: [03-react-docs-study](./03-react-docs-study)

---

## 🧭 추천 학습 흐름 (효율 루트)

1. `01-onebite-react`로 React 기본 흐름을 “손에 익히기”
2. `02-kimjeonghwan-react`로 원리 이해(중간에 막혀도 OK)
3. `03-react-docs-study`로 공식 개념/권장 패턴 확정

> ✅ “만들기 → 이해하기 → 정리하기” 사이클로 학습합니다.

---

## ✅ 학습 규칙 (내 기준)

- 주석은 **왜(WHY)** 중심
- 예제는 **작게 쪼개서** 남기기
- 같은 주제라도 성격 분리:
   - onebite = 실습/프로젝트
   - 김정환 = 원리/구현
   - 공식문서 = 정리/레퍼런스
